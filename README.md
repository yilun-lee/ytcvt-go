### This is a simple wrapper for youtube-dl implemented in golang, a practice
-----------------

###### Build 
```
go build 
./ytcvt --help
```


###### Usage 
Usage of ./ytcvt:
  -f string
        File list containing youtube link
  -o string
        The output prefix (default "/mnt/e/Youtube/")
  -u string
        The youtube url link
